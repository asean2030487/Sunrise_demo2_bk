### 技術移轉課程 Demo2 - PHP串接SQL實作註冊、登入和登出功能

* 結構說明

```text
index.php       // 首頁
register.php    // 註冊頁
login.php       // 登入頁
sql.php         // 共通sql連接php

act/insert.php  // 註冊邏輯處理
act/verify.php  // 登入邏輯處理
act/logout.php  // 登出邏輯處理
```

* 使用技術&觀念

1. php 連接MySQL
2. SQL 基本語法
3. php 基本POST表單接收    
4. php session操作
